class disarm_gears.frames.timeframe.Timeframe(start, length, by='day', step=1, end=None, directive='%Y-%m-%d')¶

Bases: object

__init__(start, length, by='day', step=1, end=None, directive='%Y-%m-%d')¶

Define a timeframe for a model.

Parameters

  • start – Initial date of the timeframe. Date encoded as: integer, float, string or datetime. Can be a None value if end is specified.

  • length – Length of the timeframe. Positive integer.

  • by – Units of time in which the frame is measured. String, one of ‘day’, ‘month’, ‘year’.

  • step – Number of units per time point. Positive integer.

  • end – Last date of the timeframe (optional and only used if start is None). Date encoded as: integer, float, string or datetime.

  • directive – Date format representation (default ‘%Y-%m-%d’). String.
