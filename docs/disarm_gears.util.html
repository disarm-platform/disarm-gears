disarm_gears.util.binomial_to_bernoulli.binomial_to_bernoulli(n_positive, n_trials, X=None, aggregated=True)¶

Takes a set binomial observations and returns a set of weighted bernoulli observations.

Parameters

  • n_positive – Number of successes in a Binomial experiment. Numpy array, shape = [n, ]

  • n_trials – Number of trials in a Binomial experiment. Numpy array, shape = [n, ]

  • X – Features associated to n_positive (optional). Numpy array, shape = [n, ] or [n, d] (default None).

  • aggregated – Whether to aggregate repeated observations. Bool (default True).

Returns

tuple (ones-zeros array [m, ], weights array [m, ], new_X array [m, ] or [m, d])
